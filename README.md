# Simon-Says-Game
A fun and interactive Simon Says game built with HTML, CSS, and JavaScript.   Players must repeat the sequences of colors, testing memory and focus.
## Features
- Classic Simon Says gameplay
- Increasing difficulty with each level

## How to Play
1. press any Key on the keyboard to start.
2. Watch the sequence of colors.
3. Repeat the sequence by clicking the buttons.
4. Each round adds a new step — see how far you can go!
